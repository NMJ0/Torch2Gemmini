# MLP MNIST Directory

This folder contains a  MNIST MLP pipeline from training to MLIR lowering and RISC-V/Gemmini execution in an FPGA.

## Files

- `Makefile`: Build and run targets for MLIR generation, lowering, linking, and execution (Spike/FPGA).
- `train.py`: Trains the MLP on MNIST and exports quantized parameters.
- `mnist_quantized.json`: Saved quantized model weights/biases used by the MLIR generator (generated by `train.py`).
- `image.py`: generates the input image .
- `generate_mlir.py`: Creates the MLP MLIR module from model parameters/input setup.
- `wrapper.c`: C entry/wrapper used when linking generated object code.
- `input.bin`: image input generated by `image.py`
- `input_data.h`: input.bin converted to c header file using
  ```bash
  xxd -i input.bin > input_data.h
  ```
